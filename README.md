# sao-create-hypernova-angular

> Hypernova Angular Boilerplate

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao create-hypernova-angular my-project
```

### From git

```bash
sao marconi1992/create-hypernova-angular my-project
```
